📦 Overview

Windows-Defender-Disabler helps you remove or disable Microsoft Defender Antivirus from your Windows computer. It cleans your system of the built-in security software. This tool modifies registry keys, stops background telemetry services, and turns off SmartScreen protection. ❓ Frequently Asked Questions

Does this permanently remove Defender?

Yes, the scripts perform deep system changes to ensure the service remains disabled. You can reverse these changes if you choose to reinstall or enable the features later.

Is it safe?

The scripts modify core system settings. Make sure you back up important data before you proceed. Disabling your antivirus software changes the security posture of your computer and exposes your system to external threats.

If the script does not start, check your local permissions. Some system policies might block script execution. Ensure that your user account has clear ownership of the folder where the tool resides. Restart your computer after the process finishes to ensure all registry changes apply correctly.

The tool works by editing the Windows Registry. It targets specific keys responsible for the Defender service. It also terminates tasks related to telemetry and reporting. These actions remove the hooks that keep the antivirus active in the background. It effectively silences alerts and background monitoring.
